# Task 1: Dataset Understanding

## Business Objective

The objective of this analysis is to identify the factors that are most strongly associated with monthly sales performance across retail stores. The insights from the regression analysis will help leadership make decisions related to marketing investment, inventory management, staffing, discount strategies, and store operations.

---

## Business Problem Summary

The leadership team of a retail chain wants to understand which business factors are most strongly associated with monthly sales performance across stores. The goal is to use regression analysis to identify key sales drivers and support decisions related to marketing investment, inventory management, discount strategies, staffing allocation, and store operations.

This project applies simple and multiple linear regression models to evaluate relationships between monthly sales and various business variables.

---

# Dataset Description

The dataset contains monthly store-level performance data for a retail chain.

The data includes information related to:

* Monthly sales
* Marketing spend
* Customer footfall
* Discount levels
* Staff count
* Inventory availability
* Customer ratings
* Competitor proximity
* Store region
* Store type
* Holiday indicators

The dataset is stored in:

```text
data/business_regression_data.xlsx
```

# Dependent and Independent Variables
## Dependent Variable

The dependent variable (target variable) for the regression analysis is:

**monthly_sales**

This variable represents the monthly sales generated by each store and is the outcome that the business wants to explain and predict.

---

## Potential Independent Variables

The following variables are potential predictors of monthly sales:

* marketing_spend
* footfall
* avg_discount_pct
* staff_count
* inventory_availability_pct
* competitor_distance_km
* customer_rating
* holiday_flag
* region
* store_type

These variables may influence sales performance either directly or indirectly.

---

## Numerical Variables

The dataset contains the following numerical variables:

| Variable                   | Description                                      |
| -------------------------- | ------------------------------------------------ |
| monthly_sales              | Monthly sales revenue                            |
| marketing_spend            | Marketing expenditure                            |
| footfall                   | Number of customer visits                        |
| avg_discount_pct           | Average discount percentage offered              |
| staff_count                | Number of employees in the store                 |
| inventory_availability_pct | Percentage of inventory available                |
| competitor_distance_km     | Distance to nearest competitor                   |
| customer_rating            | Average customer rating                          |
| holiday_flag               | Indicator variable (0 = No Holiday, 1 = Holiday) |

---

## Categorical Variables

The dataset contains the following categorical variables:

| Variable   | Description                    |
| ---------- | ------------------------------ |
| region     | Geographic region of the store |
| store_type | Type of retail store           |

These variables cannot be used directly in regression and must be converted into dummy variables.

---

## Variables Requiring Cleaning or Transformation

The following variables may require preparation before regression analysis:

### region

* Requires dummy variable creation.
* One category will be selected as the reference category.

### store_type

* Requires dummy variable creation.
* One category will be selected as the reference category.

### holiday_flag

* Should be verified to ensure values are limited to 0 and 1.

### Numerical Variables

* Check for missing values.
* Check for outliers and unrealistic values.
* Verify data types are numeric.

---

## Variables That May Not Be Useful for Regression

### store_id

* Serves only as a unique identifier.
* Does not provide meaningful business information for predicting sales.
* Will be excluded from regression models.

### month

* Primarily used for record identification and time tracking.
* Not included in the initial regression models unless transformed into seasonal indicators.

## Summary

The analysis will use monthly_sales as the dependent variable and evaluate both numerical and categorical predictors. Categorical variables will be converted into dummy variables before running regression models. Variables such as store_id will be excluded because they do not provide predictive business value. The prepared dataset will then be used for simple and multiple regression analysis to identify the strongest drivers of sales performance.
